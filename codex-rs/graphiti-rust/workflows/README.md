# Agentic Coding工作流模板

这些模板展示了如何将Graphiti MCP服务集成到Claude Code的日常开发工作流中，实现智能知识管理。

## 工作流模板列表

1. **新功能开发流程** (`new-feature.md`)
2. **Bug修复流程** (`bug-fix.md`) 
3. **代码审查流程** (`code-review.md`)
4. **重构工作流程** (`refactoring.md`)
5. **知识迁移流程** (`knowledge-migration.md`)

## 核心理念

### 知识驱动开发
- 每个技术决策都有明确的记录和reasoning
- 代码模式和最佳实践的积累和复用
- 项目经验的结构化保存和检索

### 上下文感知编程
- 基于历史经验的智能建议
- 相关代码模式的自动发现
- 团队知识的无缝共享

### 持续学习循环
- 从每个feature中提取可复用的知识
- 不断完善团队的编程实践
- 基于数据的决策优化

## 使用说明

每个工作流模板包含：
- 明确的步骤定义
- MCP工具调用示例
- 知识管理最佳实践
- 团队协作指导

选择适合的工作流模板，按照步骤执行，将开发过程中的知识自动转化为团队资产。
